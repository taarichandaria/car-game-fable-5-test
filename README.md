This game is a test of Claude Fable 5's capabilities. A classic problem I like to try with new models is to get them to build a 3D web-based game from the prompt: "Can you please code a 3D car game where I can drive a car around a city at sunset. Make the graphics beautiful." While simple, many models stumble at this due to the need for a degree of spatial reasoning.

Fable 5 successfully one shotted the implementation in less than 10 minutes on Claude Code with no graphical or functional mistakes. This is the first time this has happened with a frontier model.
